# 리뷰 스퀘어 생성기

리뷰 텍스트를 입력하면 예쁜 말풍선 형태의 정사각형 카드로 만들어주는 프로그램입니다.

## 기능

✨ **주요 기능**
- 리뷰 텍스트 입력
- 별점 선택 (1~5점)
- 작성자 이름 추가 (선택)
- 실시간 미리보기
- PNG 이미지로 다운로드

## 사용 방법

1. `index.html` 파일을 웹 브라우저로 열기
2. 리뷰 텍스트를 입력란에 붙여넣기
3. 별점 선택 (1~5개)
4. 작성자 이름 입력 (선택사항)
5. "카드 생성하기" 버튼 클릭
6. 미리보기 확인 후 "이미지로 다운로드" 버튼으로 저장

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)
- html2canvas (이미지 다운로드용)
